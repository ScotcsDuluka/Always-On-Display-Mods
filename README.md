> [!IMPORTANT]
> **Runtime Dependency:** This module requires **Root access** installed on your system via **Magisk**. The module will not work without it.<br><br>
> **OS Requirement:** **HyperOS 1.0, 2.0, or 3.0** is required. It will not work on older MIUI versions.

> [!CAUTION]
> **Brick Risk:** It is highly recommended to **back up your original `AOD.apk`** before installing this module. If something goes wrong, you will need this file to restore your system.

> [!TIP]
> For the best experience, make sure to reboot your device *twice* after installing the module via Magisk Manager to ensure the overlay applies correctly.

## Supported Devices & Version
- [X] HyperOS 1.0
- [X] HyperOS 2.0
- [X] HyperOS 3.0
- [ ] Older MIUI versions (Not Supported)
